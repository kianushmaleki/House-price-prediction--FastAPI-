# House Price API

This project provides an API for predicting house prices.

## Project Structure

- `main.py`: Entry point for running the API server.
- `api.py`: Contains the API endpoints and logic.
- `schemas.py`: Defines data schemas used by the API.
- `model_metadata.json`: Stores metadata about the machine learning model.

## Getting Started

### Prerequisites
- Python 3.8+
- Required packages (see below)

### Installation
1. Clone the repository or download the source code.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   *(Create a `requirements.txt` if not present, listing all dependencies.)*

### Running the API
Run the following command:
```bash
python main.py
```

The API will start and listen for requests.

## Usage
- Send requests to the API endpoints as defined in `api.py`.
- Refer to `schemas.py` for request/response formats.
